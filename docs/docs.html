Installation

Open your terminal and run:

pip install mustel

Works on Windows, macOS, and Linux. Requires Python 3.8+.

💡 Tip: If you have multiple Pythons, install mustel with the one you use most often.

🔍 Environment Commands

Inspect your Python setup and find where packages are installed:

mustel

List all packages in current Python

mustel all

Find every Python on your system

mustel check <pkg>

Check if a package exists anywhere

mustel diff

Compare packages across Pythons

mustel updates

Show outdated packages

mustel install <pkg>

Install and confirm which Python

🛠️ Maintenance & Health

Keep your Python environment clean and healthy:

mustel doctor

System health check

mustel clean

Clear cache and .pyc junk

mustel upgrade

Mass upgrade packages

mustel upgrade --safe

Upgrade (skip major versions)

📦 Virtual Environment Manager

No more memorizing activation commands! mustel makes venvs simple.

😕 Common Venv Problems mustel Solves

  • "How do I create a venv?" → mustel venv new
  • "Is my venv active?" → mustel venv shows status
  • "Where are all my venvs?" → mustel venv list finds them
  • "My venv is broken!" → mustel detects and helps you fix it
  • "How do I delete it safely?" → mustel venv destroy
mustel venv

Visual dashboard with status

mustel venv new

Create .venv in current folder

mustel venv list

Find all venvs on your system

mustel venv destroy

Delete venv with confirmation

❓ FAQ

Will mustel change my packages?

No. mustel only reads information. It never deletes or modifies anything (except mustel install which uses pip to install, and mustel venv destroy which asks for confirmation).

What Python versions work?

Python 3.8 and newer.

What if my venv is "broken"?

This happens when a venv was created on a different machine (e.g., synced via OneDrive). mustel detects this and shows you how to fix it: mustel venv destroy then mustel venv new.

Why "mustel"?

Named after the Mustelidae family — otters, weasels, and ferrets. Small animals that are great at finding things in hidden places! 🦦
